How to build a webpage with HTML and CSS.

To build a webpage using HTML And CSS open a text editor. Use notepad for Windows, and TextEdit for Mac. Extra steps for Mac users, after opening TextEdit, click format, click Make plain text. On your desktop make a new folder name it "root". Save your file with the name "index.html" in the root folder.

You are ready to start typing your HTML code. I your index.html file type the following. 

  <!DOCTYPE html>
  <html>
  <head>
    <meta charset="utf-8">
    <title> My first page</title>
  </head>
  <body>
	<h1>Trying HTML and CSS</h1>
	<p> This is my first webpage using HTML and CSS.</p>
  </body>
  </html>

To add your CSS style using external file you do the following:
Add the link to your html file (see below). 

  <head>
    <meta charset="utf-8">
    <title> </title>
    <link rel="stylesheet" type="text/css" href="style.css">	  
  </head>

In your CSS file type the following:

Body{
	background-color: slategray;
	color: whitesmoke;
	text-align: center;
}
